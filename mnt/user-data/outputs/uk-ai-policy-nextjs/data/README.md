# Data Directory

## Required File

Place your **uk_ai_policy_powerbi_ready.csv** file here.

This is the CSV file generated by your Python analysis script (`export_for_powerbi_updated.py`).

## File Location

Copy from your original project:
```bash
cp /path/to/old/project/uk_ai_policy_powerbi_ready.csv ./data/
```

## File Requirements

The CSV should have these columns (from your existing Python scripts):
- dept
- dept_group
- title
- published_date
- year, month, quarter, year_month
- relevance_score
- priority_category
- requires_action
- policy_type
- business_impact
- sector_focus
- ai_application
- stage
- audience
- ai_summary
- primary_topic
- key_topics
- And more...

## Important Notes

1. **File name must be exact**: `uk_ai_policy_powerbi_ready.csv`
2. **Format**: CSV (comma-separated values)
3. **Encoding**: UTF-8
4. **Size**: Should be around 900+ rows based on your analysis

## Updating Data

When you have new data:
1. Run your Python scripts to generate updated CSV
2. Replace the CSV file here
3. Refresh your browser - the dashboard will load the new data automatically!

No code changes needed - the API reads directly from this CSV file.
